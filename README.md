# CycloneIntensity
# Cyclone Intensity Prediction using Vision Transformer (ViT)

## Overview

This project focuses on predicting cyclone intensity from satellite images using a Vision Transformer (ViT) model. The objective is to support early weather analysis and improve disaster preparedness through deep learning techniques.

---

## Features

* Satellite image-based cyclone analysis
* Vision Transformer (ViT) model implementation
* Regression-based intensity prediction
* Performance evaluation using MAE and R² score
* Data preprocessing and augmentation

---

## Model Details

* Model: Vision Transformer (ViT)
* Task: Regression (Cyclone Intensity Prediction)
* Evaluation Metrics:

  * Mean Absolute Error (MAE): 9.12
  * R² Score: 0.84
